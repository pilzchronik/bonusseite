Bonusseite – Clean Starter

Enthält ein minimales, sauberes Setup für GitHub Pages (Jekyll Minima):
- _config.yml mit Base-URL /bonusseite und nur index.md in der Header-Navigation
- index.md (Start) mit Liste der neuesten Posts
- _posts/… ein Beispiel-Post (Dörnthal)
- downloads.md (nicht in Header)

Upload: Dateien/Ordner entpacken und per "Upload files" ins leere Repo schieben.
