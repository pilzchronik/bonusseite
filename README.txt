Pilz-Chronik – Bonusseite (Finales Update 2025)
=====================================================

Dieses Archiv enthält die vollständig vorbereitete Bonusseite mit Blogsystem,
Design, Download-Ordner und allen Anleitungen.

INSTALLATION
-------------
1. Entpacken Sie dieses Archiv.
2. Laden Sie den gesamten Inhalt in Ihr GitHub-Repository unter /bonusseite/ hoch.
3. Prüfen Sie, dass die Struktur wie folgt aussieht:

/bonusseite/
  ├── index.md
  ├── downloads.md
  ├── _posts/
  ├── assets/
  │   ├── custom.css
  │   └── pdf/
  │       ├── Pilz-Chronik_Band1_Leseprobe.pdf
  │       └── Errata_2025.pdf
  ├── Anleitungen/
  │   ├── Pilzchronik_Bonusseite_Download-Anleitung.pdf
  │   ├── Pilzchronik_Bonusseite_Download-Kurz_Design.pdf
  │   └── Pilzchronik_Bonusseite_Errata-Kurz_Design_v2.pdf
  ├── custom_demo.html
  └── README.txt

NACH DEM UPLOAD
----------------
- GitHub Pages baut die Seite automatisch neu auf.
- Die Blogeinträge erscheinen auf der Startseite unter "Neueste Beiträge".
- Die Downloads sind im Menü "Downloads" erreichbar.
- Designanpassungen können Sie in assets/custom.css vornehmen.

Hinweis (Stand November 2025):
Die neue index.md wurde korrekt eingespielt, wird aber auf GitHub Pages derzeit noch nicht angezeigt.
Ursache vermutlich Caching bzw. Build-Verzögerung bei Pages.
Nächster regulärer Commit (z. B. neuer Post) sollte automatischen Rebuild auslösen.
Keine inhaltliche oder syntaktische Korrektur nötig.
